# AGHub Documentation

New to AGHub? Follow these pages in order:

1. [Getting Started](getting-started.md): account setup, first login, and cluster initialization.
2. [Research Drive](research-drive.md): move data between your computer and AGHub.
3. [Installing Software](installing-software.md): add packages with Conda or containers.

## Fast path

If you are setting up AGHub for the first time, most users can finish with this order:

1. Complete all steps in **Getting Started**.
2. Configure Research Drive and test one file transfer.
3. Install extra software only if needed.
